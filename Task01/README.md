# Структура файлов данных

## ratings.csv
- userId - идентификатор пользователя
- movieId - идентификатор фильма  
- rating - оценка (от 0.5 до 5)
- timestamp - временная метка

## movies.csv
- movieId - идентификатор фильма
- title - название фильма
- genres - жанры

## tags.csv
- userId - идентификатор пользователя
- movieId - идентификатор фильма
- tag - тег
- timestamp - временная метка
